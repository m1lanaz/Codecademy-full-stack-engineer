Front End Development Cheatsheet

Remembering everything you learn can become exhausting, so this cheatsheet summarises everything you've learnt throughout all of the codecademy front end development course!

  • Basic CSS
  • Display and positioning

Basic CSS

Name property Description
Font size font-size Changes the font size for the selected element
Background color background-color Changes the background color
important rule !important Overrides every other declaration USE WITH CAUTION
Opacity opacity Changes the opacity, ranges from 0 to 1
Font weight font-weight Changes the font weight, can use the keywords such as bold
Text align text-align Aligns the text to a selected side; left, right or center
foreground text color color changes text color
background image background-image Changes background image, need to use url, url("moon.jpg")
font family font-family Changes the font family of the text

Display and Positioning

Name property Description
z-index property z-index Changes which elements can overlap, The value with the highest will be at the front and the lowest will be at the back
Positioning; fixed position: fixed; The element stays in that position in the place, even with scrolling.
display property display: block/ inline/ inline-block; Block elements take up the full width of their container, inline elements take up as little space as possible (cannot have height and width adjusted), inline-block can appear next to each pther and can have their height and width adjusted
positioning: absolute position: absolute; Removes the element out of the document flow
positioning: relative position: relative; Positions it relative to where it would've been on the page
float float Determines how far left or how far right an element should float
Clear clear How an element should behave when it bumps into another element
